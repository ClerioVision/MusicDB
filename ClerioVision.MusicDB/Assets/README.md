# Asset Files

This directory should contain the following image assets for the WinUI 3 application:

## Required Assets

1. **Square44x44Logo.png** - 44x44 pixels
   - Small tile logo
   
2. **Square150x150Logo.png** - 150x150 pixels
   - Medium tile logo
   
3. **Wide310x150Logo.png** - 310x150 pixels
   - Wide tile logo
   
4. **StoreLogo.png** - 50x50 pixels
   - Store listing logo
   
5. **SplashScreen.png** - 620x300 pixels
   - Application splash screen

## Creating Assets

You can create these assets using:
- Adobe Photoshop
- GIMP (free)
- Microsoft Paint
- Online image editors

### Recommended Colors
- Primary: #0078D4 (Windows Blue)
- Background: Transparent PNG
- Icon: White or contrasting color

### Design Guidelines
Follow the Windows App Design Guidelines:
https://learn.microsoft.com/en-us/windows/apps/design/
